## 为什么 CN2 GIA 比 CN2 GT 差距这么大？

说个简单的区别方便理解。

CN2 GT 叫 Global Transit，"半程 CN2"，从国内出境的节点走 CN2 网络，但进入中国之后走的是普通 163 骨干网。换句话说，拥堵最严重的那一段，GT 没帮你优化。

CN2 GIA 叫 Global Internet Access，全程 AS4809，进出都走 CN2。拥堵的那段，GIA 走的是独立通道，带宽冗余更充足，晚高峰不降速。

用下来最直观的差别就是：同样晚上 8-11 点，GIA 的速度和白天基本没什么区别，GT 就开始肉眼可见的慢。

这也是为什么搬瓦工最终选择砍掉 GT、保留 GIA 的原因——GT 在实际使用中的体验已经跟普通线路差不多，没有继续维护的意义。
